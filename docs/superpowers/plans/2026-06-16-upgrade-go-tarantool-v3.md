# Upgrade go-tarantool v2 → v3 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Migrate `github.com/tarantool/go-tarantool/v2` to `github.com/tarantool/go-tarantool/v3` throughout the project, updating import paths, interface definitions, and test helpers to match the v3 API.

**Architecture:** The project uses v3's slimmed-down `pool.Pooler` interface (all deprecated convenience methods removed), the new pool mode constants (`ModeAny`, `ModeRW`), and the changed `MockDoer` builder pattern for tests. The mock in `mocks/pool.go` is regenerated by moq from `mocks/gen.go` after the interface is updated.

**Tech Stack:** Go 1.22, github.com/tarantool/go-tarantool/v3 v3.0.0, moq (mock generator), testify

## Global Constraints

- Target library: `github.com/tarantool/go-tarantool/v3 v3.0.0` (exact version)
- New indirect dependency: `github.com/tarantool/go-option v1.1.0`
- Remove `github.com/tarantool/go-tarantool/v2 v2.3.2` entirely from go.mod
- Pool mode constants rename: `pool.ANY` → `pool.ModeAny`, `pool.RW` → `pool.ModeRW`
- `pool.Pooler.Do()` return type changes from `*tarantool.Future` to `tarantool.Future` (interface)
- `pool.Pooler.Close()` / `CloseGraceful()` return type changes from `[]error` to `error`
- `test_helpers.NewMockDoer` changes from constructor-with-args to a builder pattern (`AddResponseRaw`, `AddResponseError`)
- Testify updated to v1.11.1 (pulled in by v3 transitively)

package mocks

import (
	"context"
	"github.com/tarantool/go-tarantool/v3"
	"github.com/tarantool/go-tarantool/v3/pool"
	"time"
)

//go:generate moq -out pool.go . Pooler

// TopologyEditor is the interface that must be implemented by a connection pool.
// It describes edit topology methods.
type TopologyEditor interface {
	Add(ctx context.Context, instance pool.Instance) error
	Remove(name string) error
}

// Pooler is the interface that must be implemented by a connection pool.
type Pooler interface {
	TopologyEditor

	ConnectedNow(mode pool.Mode) (bool, error)
	Close() error
	CloseGraceful() error
	ConfiguredTimeout(mode pool.Mode) (time.Duration, error)
	NewPrepared(expr string, mode pool.Mode) (*tarantool.Prepared, error)
	NewStream(mode pool.Mode) (*tarantool.Stream, error)
	NewWatcher(key string, callback tarantool.WatchCallback,
		mode pool.Mode) (tarantool.Watcher, error)
	Do(req tarantool.Request, mode pool.Mode) tarantool.Future
}
